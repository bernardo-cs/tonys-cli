tonys.

An agent-friendly command-line wrapper around the TonieCloud API. Send audio — local files, stdin, or YouTube links — to your kids' creative tonies, with automatic format conversion and loudness normalization.

View on GitHub Download Install
$ brew install bernardo-cs/tap/tonys

Why "agent-friendly"?

Designed to be driven by scripts, bots, and LLM agents — not just humans.

◆Stable JSON everywhere

--json on any command. Clean data on stdout, structured errors on stderr — pipe straight into jq.

◆Meaningful exit codes

0 ok · 2 usage · 3 auth · 4 not-found · 1 other. Branch on failures reliably.

◆Self-describing

tonys schema emits a JSON tree of every command and flag, so tools can introspect the whole CLI.

◆Non-interactive auth

Credentials from env vars; the token is cached 0600 and auto-refreshed. No prompts, ever.

◆Nothing out of reach

tonys raw <METHOD> <path> hits any endpoint directly when a convenience command doesn't exist yet.

◆Audio pipeline built in

Auto-convert unsupported formats with ffmpeg and EBU R128 loudness-normalize so chapters don't jump in volume.

Install

A self-contained Go binary — no runtime to install.

Homebrew (macOS / Linux)

brew install bernardo-cs/tap/tonys

With Go

go install github.com/bernardo-cs/tonys-cli/cmd/tonys@latest

Prebuilt binary

# grab a tarball from Releases, verify, install
curl -fsSLO .../tonys_0.1.0_darwin_arm64.tar.gz
shasum -a 256 -c checksums.txt --ignore-missing
tar xzf tonys_0.1.0_darwin_arm64.tar.gz
sudo mv tonys /usr/local/bin/

Linux packages: native .deb, .rpm, and .apk builds (amd64 & arm64) are attached to every release — apt install ./tonys_*.deb, dnf install ./tonys_*.rpm, …

Optional tools: ffmpeg (conversion & normalization) and yt-dlp (YouTube import) are not bundled — install them only if you need those features. Run tonys doctor to check your environment.

Quick start

Tables by default; add --json for machine output.

# point it at your own account
export TONIE_USERNAME="you@example.com" TONIE_PASSWORD="secret"

tonys tonie list
tonys upload "Erna-Tonie" bedtime.mp3 --title "Bedtime story" --wait

# pipe from stdin, normalize to a consistent volume
cat song.mp3 | tonys upload "Erna-Tonie" - --normalize match

# import from YouTube (needs yt-dlp)
tonys yt "Erna-Tonie" "https://youtu.be/XXXX"

# discover the whole interface as JSON
tonys schema | jq .

Commands

tonys schema is the always-current reference; here's the human summary.

me / configthe logged-in user · backend config & upload limits
authlogin / status / logout — manage the cached token
household listlist households
tonielist / get / rename creative tonies
uploadupload a file (or -) as a new chapter — the main command
chapterlist / add / add-existing / rm / rename / move / sort / clear
ytimport a YouTube video or playlist (needs yt-dlp)
loudnessmeasure files / inspect the per-tonie loudness ledger
rawcall any TonieCloud endpoint directly
doctorcheck dependencies, credentials and writable paths
schemamachine-readable description of all commands
